# Front-end projektni zadatak

## Features
- Kompletno responzivna web stranica
- Preusmjeravanje na ostale stranice
- Validacija forme
- Kreiranje i pohranjivanje tokena u localStorage
- AJAX call - dohvaćanje vremenskih podataka 

## Stack
* HTML
* CSS,
* Vanilla JavaScript
* jQuery (za AJAX call)

## Setup
1. Preuzet ZIP file i ekstraktati sadržaj na računalo
2. Otvoriti projekt u code editoru
3. Otvoriti prozor za ekstenzije i instalirati Live Server ekstenziju (ako nije instaliran)
4. Otvoriti Landing folder te kliknuti desni klik na index.html i Open with Live Server

## Note

Zadatak je izrađen prema zadanim uputama iz dokumentacije koristeći JS i JsQuery, stoga nije potreban nikakav dodatan build niti dodatne instalacije.
